\subsection{Detailed Description}
This class implements a simple driver for the B\+N\+O055 Adafruit I\+MU. 

This I\+MU talk to the C\+PU over I\textsuperscript{2}C.

\# An example of how to use this driver\+: \#
\begin{DoxyCode}
1 \textcolor{comment}{#    imu = BNO055.bno055 (pyb.I2C (1, pyb.I2C.MASTER, baudrate = 100000), const(0x28))}
2 \textcolor{comment}{#    imu.sys\_status ()}
3 \textcolor{comment}{#    imu.sys\_error ()}
4 \textcolor{comment}{#    imu.get\_euler\_pitch ()     # or other data...}
5 \textcolor{comment}{#    }
\end{DoxyCode}
 The example code works for a B\+N\+O055 on a Adafruit\textsuperscript{TM} breakout board.
