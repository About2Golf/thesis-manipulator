Detailed Description

This class implements a simple driver for the BNO055 Adafruit IMU.

This IMU talk to the CPU over I2C.

# An example of how to use this driver: #

1 # imu = BNO055.bno055 (pyb.I2C (1, pyb.I2C.MASTER, baudrate = 100000), const(0x28))
2 # imu.sys_status ()
3 # imu.sys_error ()
4 # imu.get_euler_pitch () # or other data...
5 #

The example code works for a BNO055 on a AdafruitTM breakout board.
