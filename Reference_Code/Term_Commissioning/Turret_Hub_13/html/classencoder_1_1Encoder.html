Initializes the pins and timer channels for an encoder object.

To create PB6 and PB7 Encoder reader:
pin1 = pyb.Pin.board.PB6 # Pin A
pin2 = pyb.Pin.board.PB7 # Pin B
timer = 4

To create PC6 and PC7 Encoder reader:
pin1 = pyb.Pin.board.PC6 # Pin A
pin2 = pyb.Pin.board.PC7 # Pin B
timer = 8

Parameters
timerSpecifies the timer for the encoder
pin1First pin (A) used to read the encoder
pin2Second pin (B) used to read the encoder
