Detailed Description

This defines the task function method for an encoder.

The encoder passes its data via a shared variable with another task. To create an instance of this task class (example):

create encoder position shared variable

pan_position = task_share.Share ('i', thread_protect = False, name = "Share_0_pan_position")

create encoder 1 task object

pan_encoder = Encoder_Task(pan_position, 4, pyb.Pin.board.PB6, pyb.Pin.board.PB7)

create task1 function

task1 = cotask.Task (pan_encoder.enc_fun(), name = 'Task_1', priority = 5, period = 2, profile = True, trace = False)

append task1 to list of scheduled tasks

cotask.task_list.append (task1)
